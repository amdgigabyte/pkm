NodeJS的安装

从github上面或者nodejs官网下最新版本的node安装包，解压（版本API变动比较大，建议安装最新的版本），目前最新的稳定版本是0.4.1， 还有pre版本的0.5.2 

git clone git://github.com/ry/node.git

cd node

cd 到解压的目录

执行以下命令 

./configure

make

make install

可能过程中会提示缺少openssl等，使用软件仓库添加相应的软件就行(sudo apt-get install libssl-dev) 

sudo apt-get install libssl-dev
